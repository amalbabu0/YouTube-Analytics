YouTube-Analytics

A sleek and interactive Streamlit web app that lets you explore YouTube videos by keyword and visualize essential stats like views, likes, and comments.


 Installation

Clone the Repository:

    git clone https://github.com/amalbabu0/youtube-stats-explorer.git
    cd youtube-stats-explorer

Create a Virtual Environment (Optional but Recommended):

    pip install -r requirements.txt

 YouTube API Key Setup

change DEVELOPER_KEY = "YOUR_YOUTUBE_API_KEY" with your key

 TO Run the App

    streamlit run streamlit_app.py

Then go to:

http://localhost:8501
